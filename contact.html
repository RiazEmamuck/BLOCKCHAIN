Cookies

Pour le développeur de ce site, votre expérience est une priorité. C’est pourquoi il utilise des cookies au titre de notre intérêt légitime à améliorer votre expérience utilisateur, réaliser des statistiques.

En poursuivant votre navigation, vous acceptez le dépôt de cookies, pour accéder à certaines de vos données personnelles liées à votre navigation sur ce site (telles que l’IP, les pages visitées…).
Elles sont utilisées pour mesurer leur performance et pour vous permettre de partager sur les réseaux sociaux. Vous pouvez changer d’avis à tout moment en cliquant sur « Tous droits réservés / Mentions légales » figurant en bas de chaque page.

Ces sites web pourraient collecter des données sur vous, utiliser des cookies, embarquer des outils de suivis tiers, suivre vos interactions avec ces contenus embarqués si vous disposez d’un compte connecté sur leur site web.

Durées de stockage de vos données

Si vous envoyez un message via le formulaire contact, le message et vos informations personnelles sont conservés momentanément dans une boîte mail.
Cela permet de vous recontacter afin que vous puissiez vous inscrire pour une formation ou un stage.

Pour les utilisateurs et utilisatrices qui s’enregistrent sur notre site (si cela est possible), nous stockons également les données personnelles indiquées dans leur profil.
Tous les utilisateurs et utilisatrices peuvent voir, modifier ou supprimer leurs informations personnelles à tout moment (à l’exception de leur nom d’utilisateur·ice). Les gestionnaires du site peuvent aussi voir et modifier ces informations.

Vos droits sur vos données

Si vous avez un compte ou si vous avez laissé des messages sur le site, vous pouvez demander à recevoir un fichier contenant toutes les données personnelles que nous possédons à votre sujet, incluant celles que vous nous avez fournies.

Vous pouvez également demander la suppression des données personnelles vous concernant. Cela ne prend pas en compte les données stockées à des fins administratives, légales ou pour des raisons de sécurité.
